# MklinkTool - Windows 符号链接管理工具

一个基于 Electron 的 Windows 桌面应用，提供图形化界面管理 mklink 创建的符号链接、目录联接和硬链接。支持跨设备使用，打包为便携版 EXE，无需安装即可运行。

## 功能特性

- **查看链接**：自动扫描系统中常见的符号链接、目录联接、硬链接
- **创建链接**：支持三种链接类型
  - 符号链接 (Symbolic Link) - 文件或目录
  - 目录联接 (Junction) - 仅目录，无需管理员权限
  - 硬链接 (Hard Link) - 仅文件，同盘符
- **修改链接**：修改链接的目标路径、位置、名称、类型
- **删除链接**：安全删除链接（仅删除链接本身，不影响目标文件）
- **目标检测**：自动检测链接目标是否有效
- **搜索过滤**：按名称、路径、目标快速搜索
- **资源管理器集成**：在资源管理器中打开链接或目标位置

## 使用方法

### 开发模式运行

```bash
npm install
npm start
```

### 打包

```bash
# 同时生成安装版和便携版
npm run build

# 仅生成便携版 (单文件 EXE，适合跨设备使用)
npm run build:portable

# 仅生成安装版 (NSIS 安装程序)
npm run build:nsis
```

打包后的文件位于 `dist/` 目录。

### 跨设备使用

推荐使用 **便携版 (Portable)**：
1. 运行 `npm run build:portable`
2. 将 `dist/MklinkTool-Portable-1.0.0.exe` 复制到任意 Windows 设备
3. 右键 → 以管理员身份运行（创建符号链接需要管理员权限）

## 注意事项

- 创建**符号链接 (Symbolic Link)** 需要管理员权限
- 创建**目录联接 (Junction)** 和**硬链接 (Hard Link)** 不需要管理员权限
- 程序已配置为启动时请求管理员权限
- 删除链接仅删除链接本身，不会删除目标文件/目录内容
- 修改链接的实现方式为"先删除后创建"，操作前请确认数据安全

## 技术栈

- Electron 30
- 原生 HTML/CSS/JavaScript（无框架依赖）
- electron-builder 打包

## 链接类型说明

| 类型 | 命令 | 适用 | 跨卷 | 需管理员 |
|------|------|------|------|----------|
| 符号链接 | `mklink /D` 或 `mklink` | 文件/目录 | 是 | 是 |
| 目录联接 | `mklink /J` | 仅目录 | 是 | 否 |
| 硬链接 | `mklink /H` | 仅文件 | 否 | 否 |
